# Luke-Util-Scripts
個人的に使いたくて作ったスクリプトの置場です  
適当に増えたりするかもしれません  

# 各スクリプトの説明
### App_List_Generator  
Windowsにインストールされているアプリケーション一覧を取得するバッチファイル  

### Restart_And_UEFI_Startup  
Windowsを再起動してUEFIのセットアップ画面に移動するバッチファイル  
警告なしに再起動するので注意  

### Win10_Style_RightClickMenu  
Windows11の右クリックメニューを従来版に戻すレジストリファイル  
適用後は再起動が必要です
